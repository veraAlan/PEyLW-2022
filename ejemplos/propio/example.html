Basic Tags

Text
  1. <h1> to <h6> : HTML Heading
  2. <p> : Pragraph
  3. <pre> : Preformatted text
  4. <ul> and <ol> : Lists
    • <li> : List item
  5. <dl> : Glossary
    • <dt>: Term
      • <dd> : Definition
  6. <a href="" or target="target="_blank|_self|_parent|_top nombre"> : Hypertext

Visual
  1. <img src="" alt=""> : Image tag
  2. <hr> : Horizontal rule
  3. <table> : Line break
    • <tr> : Table row
    • <td rowspan=""> : Table data
    • <th colspan=""> : Header cell

Forms

Action attribute: Specifies where to send the form-data when a form is submitted.

Method attribute: specifies the HTTP method to be used when submitting the form data. The form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post").
